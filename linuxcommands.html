Linux Commands

cd

The "cd" command changes the directory that your commands are currently referencing.
cd /var/www/html/student3

ls

The "ls" command lists all the files in a directory.
ls

mv

The "mv" command moves or renames a file or directory.
mv LinuxCommands.html Linux.html

rm

The "rm" command deletes a file.
rm file1

kill

The "kill" command terminates a process.
kill word.exe

whoami

The "whoami" command returns your user name.
whoami

nano

The "nano" command is a simple text editor.
nano pi.txt

sed

The "sed" command is a function that finds and replaces based on regular expressions.
sed 's/unix/linux/' example.txt

echo

The "echo" command returns a message or value.
echo egg
